# Netflix Clone
App simula a interface do Netflix, podendo visualizar filme em destaque com pontuação, sinopse, genero e foto de capa,
filmes recomendados, filmes em alta e por genero. 
 

##

Tecnologias utilizadas:

- JavaScript
- React JS
- CSS
- Material UI
